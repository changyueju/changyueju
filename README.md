# 北京城建畅悦居违法隔离和商品房涉嫌侵占公共绿地

## 背景介绍
畅悦居和龙樾华府同属一个物业管理区，且建筑物核准名（在住建委备案名）均为畅悦家园。 公安备案地址为， 昌平区平西府中路1号院（A区） 昌平区平西府东路1号院一区和二区（D区）。 
* 一个物业管理区（共同的公摊物业用房，位于A区，D区2号楼）
* 住建委备案名称完全一致
* 公安备案地址院址D区完全一致（A区和D区院址不一致，但仍属于同一个物业管理区）
